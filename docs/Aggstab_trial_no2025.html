Aggregate size classes

Soaking 10 g sub-samples of Lucilla’s bulk density sample of plot 20 in five different extractants to remove cementing agents, conducting Anova tests and rendering with ggthemr('fresh'), displays these plots (note compact letter display indicates significance of differences amongst extracts for the given size class):

Since calcimetry showed no SIC, we expected no HCl effect and can eliminate this analysis going forward. Not sure if H2O2 will become more significant with improved recovery? (syringe to remove supernatant below light fraction, followed by centrifuge). NaOH had significant increase in clay-silt fraction.
